<h1>Description:</h1>
<ul>
<strong>push_swap</strong> is a sorting algorithm project from the 42 curriculum.  
The goal is to sort a stack of integers using a limited set of operations , while producing the smallest possible number of instructions.
<ul>
<li>push_swap:</li> generates a sequence of instructions that sorts stack a
<li>checker (bonus):</li> verifies whether a given sequence of instructions correctly sorts the stack
</ul>
The challenge lies in designing an efficient algorithm while strictly respecting the allowed operations and constraints.
</ul>


<h1>Instructions:</h1>
<ul>
<li>Compilation :</li>
To compile the mandatory part:
<code>make</code>
<br>
To compile the bonus (checker):<code>make bonus</code>
<li>Execution :</li>
<ul>
<li>Example :</li>
<code>./push_swap 3 2 1</code>
<br>
<li>Output example:</li>
<code>sa <br>
rra</code> <br>
<li>checker (bonus):</li><code>echo -e "sa\nrra" | ./checker 3 2 1</code>
<li>output</li>
<code>OK</code><br>
<li>Or combined with push_swap:</li>
<code>./push_swap 3 2 1 | ./checker 3 2 1</code>
</ul>
</ul>
